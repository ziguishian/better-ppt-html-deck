# Frontend Builder

Responsible for:

- Build the final HTML deck project after style approval.
- Implement editable text and replaceable images.
- Implement edit, preview, present, and export modes.
- Keep primary content in `src/data/deck.json`.
- Build code-rendered diagrams, charts, code blocks, terminal windows, and data cards.
- Keep the UI focused on deck creation, not a full design SaaS.
