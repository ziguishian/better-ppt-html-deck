# Visual Style Rules

Use these rules when choosing or revising visual direction.

## Style Selection

- Formal report: restrained palette, clear grid, strong information hierarchy, low ornament.
- Pitch deck: high contrast, strong headlines, confident hero visuals, crisp data moments.
- Course/tutorial: readable typography, step-by-step structures, examples, code/terminal frames when relevant.
- Product launch: product-first composition, controlled drama, large type, polished component system.
- Social carousel: stronger cover, clear page rhythm, fewer words, bold hooks.

## Palette

- Define background, surface, primary text, secondary text, accent, warning/success when needed.
- Avoid one-note palettes dominated by a single hue unless the brand requires it.
- Limit neon/glow to a small accent role.

## Typography

- Use no more than two font families unless brand assets require more.
- Use size and weight first; use color sparingly for emphasis.
- Avoid long center-aligned paragraphs.

## Layout

- Choose one grid and keep it stable.
- Use varied slide structures across the deck: cover, section, big idea, process, comparison, data, summary.
- Make diagrams code-rendered and editable whenever possible.

## Image Language

- Use generated images only as local visual assets.
- Prefer subject-relevant images, product detail, editorial textures, or abstract background assets that do not fight the text.
